# 判断规则文档

> 

- 非单项判断条目先计算理论上限，异常/存疑 = 系数 × 理论上限

## 判断级别

| 级别 | 标准 |
|------|------|
| **异常** | 正常玩家几乎无法达成，99% 实锤作弊 |
| **存疑** | 相对极限数据，正常玩家难以达成 |

---

## 一、经济类

### 1.1 总收入

**目的:** 检查是否利用严重漏洞赚取游戏币或使用外挂作弊。优先考虑能第一时间赚钱的方式。

| 参照项 | 说明 | 阈值 |
|--------|------|------|
| 老抢劫犯罪之神 | 最高效挣钱方式 | 熟练车队约 6h，收入 1250W |
| 被带抢劫 | 佩里科 85% 分红，二人约 200W/把，15min/把 | 25 把 或 在线时间 1/10，取最小值 |
| 全产业收入 | 以版本最高赚钱速度 × 24h 周期量化挂机 | 在线 > 12h 即可开始 |
| 最快获取收入 | 不停佩岛 | 在线 > 4h 即可开始（丐版虎鲸） |

**相关数据:** 角色在线时间、总收入

---

### 1.2 收支差距

**目的:** 检查是否通过作弊手段刷钱（官网数据仅精确到小数点后一位，大额时需计算精度丢失）。

**公式:**
```
收支差距 = 总花费 + 现金 + 银行存款 - 总收入 - 黑钱/鲨鱼卡 - 精度丢失
```

**阈值:**
- 起步允许 10M
- 每在线 1 小时 +1W
- 封顶 50M
- 阶梯状提升
收支差不能大于10%
---

### 1.3 收入被隐藏

**目的:** 不当刷钱或藏匿大量应有收入，反向计算收支差距。仅存疑（无法明确是否与作弊相关）。

---

### 1.4 收入统计被修改

| 规则 | 判断逻辑 |
|------|----------|
| 差事收入被修改 | 饼图与详情统计对应项数据对不上 |
| 下注收入被修改 | 同上 |
| 车辆销售收入被修改 | 同上 |
| 其他收入被修改 | 同上 |

---

### 1.5 差事收入

**目的:** 与总收入类似，检查利用漏洞或外挂赚取游戏币。

| 参照项 | 阈值 |
|--------|------|
| 老抢劫犯罪之神 | 熟练车队约 6h，收入 1250W |
| 被带抢劫 | 佩里科 85%，二人约 200W/把，15min/把；25 把或在线时间 1/10 取最小 |
| 最快获取收入 | 在线 > 4h，丐版虎鲸 |

**相关数据:** 角色在线时间、差事收入

---

### 1.6 差事回报率

**目的:** 判断是否卡分红。假设玩家仅自行主持差事且极其大方分享收入，计算回报率。

**相关数据:** 差事收入、差事入场费

---

### 1.7 拾取收入

**目的:** 主要与刷钱挂钩。

- 存在起步值（为被刷钱袋或萌新捡挂钱袋留余地）
- 剩余额度 = 在线时间 × 理论最多拾取速率

**相关数据:** 在线时间、拾取收入

---

### 1.8 分成收入 / 给予他人

**目的:** 两项均为极早期数据，现已无法产生。按角色创建时间判断是否刷钱，按在线时长判断记录数量是否合理。

**相关数据:** 角色创建时间、角色在线时长、分成收入、给予他人

---

### 1.9 下注收入

**目的:** 判断是否通过筹码无冷却置换 BUG 获取游戏币。

---

### 1.10 卖车收入

**目的:** 判断是否使用复制车、CE 置换等方式通过卖车获取不当收入。

**相关数据:** 角色在线时长、载具和维护花费、出售载具收入

---

### 1.11 其他收入

**目的:** 目前仅含表现良好奖励（2000/游戏日，约 48min）。一般玩家很少在战局待满一整天，可轻易推算。

**相关数据:** 角色在线时长、其他收入

---

## 二、在线时长类

### 2.1 在线时长不一致

**目的:** 判断是否修改角色在线时长。短时间为修改，长时间为不一致。

**相关数据:** 角色在线时间、账号在线时间

---

### 2.2 每日在线时长

**目的:** 判断是否修改账号游戏时间。

**相关数据:** 角色创建时间、角色在线时间

---

### 2.3 应有表现良好奖励

**目的:** 通过应有表现良好奖励反推实际在线时间，检查在线时长是否被修改。

**相关数据:** 角色在线时长、表现良好奖励

---

### 2.4 应有拾取金额

**目的:** 检查是否修改击杀数和在线时长。

**相关数据:** 角色在线时长、拾取收入、角色玩家击杀数、击杀数

---

## 三、等级与经验

### 3.1 升级速度

**目的:** 判断是否修改等级。由于存在继承等级、仙人掌、收集品等高效升级方式，此项较为宽裕。

- 继承等级从宽：非本角色在线时间认为全在吃仙人掌，最多 120 级
- **不认可**竞技场转盘或脚本消星升级

**经验获取速率:**

| 方式 | 速率 |
|------|------|
| 仙人掌 | 35W 经验/h |
| 收集品 | 约 5W 经验/h（4h 20W） |
| 拉货 | 2500 经验/6min |

**仙人掌出现时间:**
```
2019/10/31, 2019/11/07, 2020/04/06, 2020/10/22,
2021/02/18, 2021/10/28, 2022/10/27, 2023/04/06,
2023/11/16, 2023/11/23, 2024/01/04, 2024/05/09,
2025/04/03, 2025/10/23, 2025/10/30
```

**相关数据:** 账号在线时间、角色在线时间、角色等级

### 4.4 PVP 爆头率

**目的:** 检查 PVP 爆头率是否过高。

- 常 PVP 玩家用枪击杀本来就少（Mk2、天煞、RPG 为主）
- 普通子弹打防弹头盔需 3 枪，打身子伤害反而更高
- **已知 BUG:** 犯罪集团更新后，战局玩家击杀数不增加记录，但爆头击杀数增加

**相关数据:** 角色玩家击杀数、角色玩家爆头击杀数

### 4.5 K/D

**目的:** 判断是否修改玩家击杀数和被玩家击杀数。

**相关数据:** 玩家击杀数、被玩家击杀数

### 4.6 玩家击杀数不一致

**目的:** 角色玩家击杀数与总玩家击杀数不一致。与"玩家击杀数被修改"的区别是差距较小。

## 五、载具类

### 5.1 陆上载具最高速度

**目的:** 检测是否修改载具最高速度。

- 挂车瞬移不会导致超速
- 已知陆上载具极速：竞速图 10km 级高空加速带，接近 **480km/h**
- 其余载具最高 ≤ **350km/h**
- 暴君倒飞着陆最高速度不高
- 存疑项额外判断最低速度（不可能游玩时间超长，最高速度仅十几）

**相关数据:** 陆上载具最高速度、驾驶过最高速的陆上载具、在线时长

---

## 六、飞车特技

### 6.1 发现特技地点

**目的:** 检查是否修改发现飞车特技地点。

- 分母固定 **50**
- 分子 +2 以上才判断异常
- 速度超标也触发

**相关数据:** 发现飞车特技地点

---

### 6.2 完成飞车特技

**目的:** 检查是否修改完成飞车特技地点。

- 数额限制与发现特技相同
- 完成数 < 发现数
- 速度超标也触发

**相关数据:** 发现飞车特技地点、完成飞车特技地点

---

## 七、其他

### 7.1 最高生存战波数

**目的:** 简单数值判断是否修改生存战波数。

**相关数据:** 最高生存战波次

---

### 7.2 汽车出口数被修改

**目的:** 判断是否修改西门战局偷车任务。

- 出口数 < 窃取数
- 最多 40min/辆

**相关数据:** 角色在线时长、载具窃取数、载具出口数

---

### 7.3 被通缉次数被修改

**目的:** 检查是否修改被通缉次数。

**约束:**
```
0 ≤ 被通缉次数 ≤ 被通缉星数 ≤ 被通缉次数 × 5
```

**相关数据:** 被通缉次数、被通缉星数

---

### 7.4 逃脱通缉次数被修改

**目的:** 检查是否修改逃脱通缉次数。

**约束:**
```
0 ≤ 逃脱通缉星数 ≤ 被通缉星数
```

**相关数据:** 逃脱通缉星数、被通缉星数

---

### 7.5 被持续通缉时间被修改

**目的:** 检查是否修改被通缉时间。

**约束:**
```
0 ≤ 五星通缉时间 ≤ 通缉时间 ≤ 角色在线时长
0 ≤ 上次被通缉时间 ≤ 最长被通缉时间 ≤ 角色在线时长
```

**相关数据:** 五星通缉持续时间、被通缉时间、上次被通缉时间、最长被通缉时间、角色在线时长

---
